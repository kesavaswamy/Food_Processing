# 🍽️ Food Processing & Product Management Application

## 📌 Project Overview
This is a web-based application developed using **ASP.NET MVC** for managing and displaying food product details.  
The application allows users to view product descriptions, manage quantity selection, and experience a clean, responsive UI.

This project is designed with a focus on:
- Clean UI
- Structured MVC Architecture
- Maintainable and scalable code

## ✨ Key Features

- 🏠 Home page with product descriptions
- ➕ Quantity increment & decrement functionality
- 📦 Product display with clean UI
- 🧾 Responsive layout using Bootstrap
- 🗄️ Database integration with SQL Server
- 🔄 MVC architecture implementation

---

## 🧠 Architecture

The application follows the **MVC (Model-View-Controller)** pattern:

- **Model** – Handles business logic and database interaction
- **View** – Razor views for UI rendering
- **Controller** – Manages request handling and user interactions
